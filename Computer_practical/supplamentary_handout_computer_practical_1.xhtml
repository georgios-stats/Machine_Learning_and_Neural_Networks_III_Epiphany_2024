The goal (roughly speaking...)

Consider a set of examples S ={ z 1 ,..., z n } of size n , with z i ∼ IID g .
Consider functions R( w ) = E z ∼ g ( ℓ ( w,z ) ) , and R S ( w ) = 1 n ∑ i=1 n ℓ ( w, z i ) where w ∈ W ⊂ R d and ℓ ( ⋅ , ⋅ ) is a loss function.
Find minimizers in the following problems w * = arg min w{ R( w ) } = arg min w{ E z ∼ g ( ℓ ( w,z ) ) } w ** = arg min w{ R S ( w ) } = arg min w{ 1 n ∑ i=1 n ℓ ( w, z i ) } Both of the following algorithms are often used to approximate w * and w **

Gradient descent algorithm

Provide:
  • A seed (initial value) w ( 0 )
  • Learning rate η t : N → R + ; a positive non-increasing function such as η t → 0 . E.g.
    • η t = t 0 /t , with t 0 >0 user specified
    • η t = something retatively tiny
For t=1,2,3,... iterate:
  1. compute w ( t+1 ) = w ( t ) - η t ∇ R S ( w ( t ) )
  2. terminate if a termination criterion is satisfied, e.g. If t ≥ T max then STOP
Return w ‾ = mean { w ( t ) }

Stochastic gradient descent algorithm

Provide:
  • A seed (initial value) w ( 0 )
  • Learning rate η t : N → R + ; a positive non-increasing function such as η t → 0 . E.g.
    • η t = t 0 /t , with t 0 >0 user specified
    • η t = something retatively tiny
  • An integer value for m ∈ { 1,...,n } , i.e. m<<n .
For t=1,2,3,... iterate:
  1. collect a random sub-sample S ˜ ={ z ˜ j ( t ) ;j=1,...,m } of size m with or without replacement from the complete data-set S
  2. compute w ( t+1 ) = w ( t ) - η t ∇ R S ˜ ( w ( t ) ) where R S ˜ ( ⋅ ) = 1 m ∑ i=1 m ℓ ( ⋅ , z ˜ i ) .
  3. terminate if a termination criterion is satisfied, e.g. If t ≥ T max then STOP
Return w ‾ = mean { w ( t ) }
